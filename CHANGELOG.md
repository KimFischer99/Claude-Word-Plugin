# Changelog

## Beta 0.1.0

- Added macOS package installer for the Word add-in and local service runtime.
- Added Claude Web account import through the local proxy.
- Added multi-account routing and custom Anthropic-compatible API settings.
- Added Word context attachment from selections, document excerpts, quotes, and pasted links.
- Added built-in summarize, humanize, and review shortcuts.
- Added uninstall flow from the settings drawer.
